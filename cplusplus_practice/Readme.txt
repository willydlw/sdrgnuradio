The dial_tone.cc example is directly from the gnuradio website.
Greatly appreciated, as well, was the CMakeLists.txt.

Had to make a few changes to the CMakeLists.txt to update it for 
a header name change.

To build any of the examples, navigate into that directory and
follow the instructions below.

To build:

1. mkdir build
2. cd build
3. cmake ../
4. make

Run the executable
